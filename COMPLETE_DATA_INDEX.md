# Complete Data Index

## Overview

This document provides an index to all publicly available data files in this repository.

## Seeds and Identities

### Sample Data (100 Seeds)

- **`100_SEEDS_AND_IDENTITIES.md`** - Human-readable table with 100 seeds, documented identities, real identities, match status, and differences
- **`100_seeds_and_identities.json`** - Machine-readable JSON format with complete data

**Contents**:
- 100 seeds (55 characters each)
- 100 documented identities (from matrix extraction)
- 100 real identities (derived from seeds)
- Match status (0 matches, 100 mismatches)
- Character differences per identity

### Complete Database (23,765 Seeds)

- **`MAPPING_DATABASE_SUMMARY.md`** - Summary of complete mapping database
- **`ALL_23765_SEEDS_SUMMARY.md`** - Complete seeds database summary

**Note**: The complete mapping database (`complete_mapping_database.json`, ~50MB) is not included in this repository due to size. The summary files above contain all essential statistics and analysis.

## Access

All files are in the repository root. The complete mapping database (`complete_mapping_database.json`, ~50MB) is not included due to size. See summary files above.

## Verification

All identities can be verified on-chain using the scripts provided:
- `scripts/verify/rpc_check.py` - On-chain verification
- `scripts/verify/control_group.py` - Control group test
- `scripts/verify/statistical_significance.py` - Statistical analysis
