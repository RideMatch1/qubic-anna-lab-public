# 100 Seeds and Identities - Complete Documentation

This document contains 100 seeds and their corresponding Qubic identities found in the Anna Matrix, including both documented identities (from matrix extraction) and real identities (derived from seeds using Qubic Wallet).

## Important Discovery

**Critical Finding**: Seeds derived using `identity.lower()[:55]` generate different Qubic identities than the original documented identities from the matrix.

- **Documented Identity**: Extracted directly from Anna Matrix (60 characters, uppercase)
- **Real Identity**: Generated by Qubic Wallet from the seed (60 characters, uppercase)
- **Match Rate**: 0% (0 matches out of 100)

This indicates the `identity.lower()[:55]` formula is an approximation, not the true transformation.

## Format

- **Seed**: 55-character lowercase string (derived from documented identity using `identity.lower()[:55]`)
- **Documented Identity**: 60-character uppercase Qubic identity extracted from matrix
- **Real Identity**: 60-character uppercase Qubic identity generated by Qubic Wallet from seed
- **Match**: Whether documented and real identities match
- **Differences**: Number of character differences between documented and real identities

## Interpretation

The matrix contains the final identities (documented), not the seeds that generate them. The `identity.lower()[:55]` formula produces valid seeds that generate different (but also on-chain) identities.

This suggests:
- The matrix may contain multiple layers of information
- The true transformation function f(Matrix) = Seed is unknown
- Both documented and real identities are valid and on-chain
